Essa é uma página HTML servida via Github Pages.

O Github Pages serve somente páginas estáticas!

O que isso significa na prática? Acesso somente a tecnológias front-end, nada de banco de dados, nem semelhantes, só HTML, Javascript e CSS...

Entrentanto, se pode ser feito com os três, então ainda dá para ser feito aqui.

Ademais, a página servida é automaticamene atualizada toda vez que há um commit na main, o processo demora no máximo 1 min. Contudo, cacheamento do browser pode fazer você ver versões antigas, para resolvir isso ative 'Desativar cache' na aba Redes do painel do desenvolvedor.

E não, não é possível mudar o endereço, nem é para mudar o nome desse arquivo, pois o ponto de entrada de serviço do Github Pages sempre será chamado 'index.html'
